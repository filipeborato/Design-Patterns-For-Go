# Design-Partners-for-Go

This project is a study of some types of design patterns for Golang. 

## Partners:
- Factory
- Generator
- Observer
- Provider

## Go Version on the project was made:
`
V1.16
`

## How to run the project:
```
go mod vendor {mainFolder}
```

```
 go build {mainFolder}
```

```
 go run main.go
```
